Movies API

allows users to search movie database by movies, directors and genres. Users can sign up and create their own favorites list

Request URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET None JSON object including data about movies
Get info about a movie based on title /movies/[title] GET None JSON object including data about a movie based on title (title, director, genre). Example: { "_id" : ObjectId("5e4962305f0c94b3f38595cc"), "title" : "Fight Club", "description" : "An insomniac office ... ", "director" : { "name" : "David Fincher", "bio" : "is an American director...", "birth" : "28.08.1962" }, "genre" : { "name" : "Drama", "description" : "Within film, television ..." }, "imageURL" : "/img/movie.jpg", "featured" : false }
Add new movie to the database /movies POST Title is required. {"title" : "American Psycho", "description" : "A wealthy New York City (…)", "director" : { "name" : "Mary Harron"}, "genre" : { "name" : "Crime”}, "imageURL" : "/img/movie.jpg"} {"_id" : ObjectId("5e4961605f0c94b3f38595c8"), "title" : "American Psycho", "description" : "A wealthy New York City (…)", "director" : { "name" : "Mary Harron"}, "genre" : { "name" : "Crime”}, "imageURL" : "/img/movie.jpg"}
Delete movie from database /movies/[id] DELETE None Message – movie removed
Add user – open account /users POST JSON object holding new user data. Required: username, password and email. {username: 'user1', password: 'password1', email: 'email1@test.com', birthday: new Date('1994-02-17')} JSON object holding user data, including ID. { "_id" : ObjectId("5e49a4cece6f8e27e88042e7"), "username" : "user1", "password" : "password1", "email" : "email1@test.com", "birthday" : ISODate("1994-02-17T00:00:00Z")}
Update username /users/[username] PUT JSON object with updated user info (now username, rest coming soon). Example: {"username" : "newusername"} text message indicating the user's information was updated
Add movie to favorite list /[username]/favorites POST JSON object data about the movie. Example: {"title": "Wild at Heart"} A text message – added to favorites
Remove from favorites list /[username]/favorites/[id] DELETE None A text message – removed from favorites
