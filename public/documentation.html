Movies API

allows users to search movie database by movies, directors and genres. Users can sign up and create their own favorites list

Request URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET None JSON object including data about movies
Get info about a movie based on title /movies/[title] GET None JSON object including data about a movie based on title (title, director, genre). Example: {"title": "Wild at Heart", "director": "David Lynch", "genre": ["road", "romance"], "image": "./img/wild_at_heart.jpg", "id": "1"}
Add new movie to the database /movies POST Title is required. {"title": "Wild at Heart", "director": "David Lynch", "genre": ["road", "romance"], "image": "./img/wild_at_heart.jpg"} {"title": "Wild at Heart", "director": "David Lynch", "genre": ["road", "romance"], "image": "./img/wild_at_heart.jpg", "id": "1"}
Delete movie from database /movies/[id] DELETE None Message – movie removed
Get information about chosen genre /genres/[name] GET None JSON object – genre name and description. Example: {"name": "road", "description": "A road movie is a film genre in which the main characters leave home on a road trip, typically altering the perspective from their everyday lives."}
Get information about chosen director /directors/[name] GET None JSON object – director name, date of birth and short bio. Example:{"name": "David Lynch", "birth": "", "death": "", "bio": ""}
Add user – open account /users POST JSON object holding new user data. Required: username, password and email. {"username": "user1", "password": "password1", "email": "user1@email.com", "birth": "21.09.1989"} JSON object holding user data, including ID. {"username": "user1", "password": "password1", "email": "user1@email.com", "birth": "21.09.1989", "id": "1"}
Update username /users/[username] PUT JSON object with updated user info (now username, rest coming soon). Example: {"username" : "newusername"} text message indicating the user's information was updated
Add movie to favorite list /[username]/favorites POST JSON object data about the movie. Example: {"title": "Wild at Heart", "director": "David Lynch", "genre": ["road", "romance"]} A text message – added to favorites
Remove from favorites list /[username]/favorites/[id] DELETE None A text message – removed from favorites
